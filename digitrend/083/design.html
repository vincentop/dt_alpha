進化的超新星 ROG Cetra True Wireless SpeedNova

真無線藍牙耳機

2022年ROG推出了品牌旗下的第一支TWS(真無線耳機)-ROG Cetra True Wireless，並獲得廣泛好評，這股成功的浪潮激發了團隊的靈感，讓ROG團隊不斷思索如何進一步精進這款優秀之作。經過約兩年的潛心研發，ROG於2024年初再次推出了完美進化的ROG Cetra True Wireless SpeedNova！接下來，我們將深入探討ROG團隊如何運用卓越的開發精神以及設計思維，讓這款備受讚譽的產品更上一層樓。

數位狂潮 第83期 品味設計／圖：ROG Official Data／設計師：ROG Kuan／統整撰寫：ROG Lovelock

/ultra-cool/

首先，讓我們從外觀設計開始說起，從圖中可明顯看出，ROG Cetra True Wireless SpeedNova相較於ROG Cetra True Wireless，在外觀的輪廓與細節上，更加凸顯其專為遊戲而生的酷玩風格，不僅在 整體外觀上更Gaming化，而且還更精緻化，ROG Cetra True Wireless SpeedNova不但讓原本體積就非常小的TWS耳機本體有了更吸睛的ROG發光LOGO以及ROG銘板等細節點綴，在充電盒內部也將ROG發光LOGO以及其周圍區域做了更細緻的亮霧交錯斜紋處理設計，並加入了FOR THOSE WHO DARE的ROG Tagline精神標語，此外，充電盒的外部亦加上細緻的高亮線條以及立體稜線，讓整體看起來不但更有精神，還帶有速度感，體現出玩家們分秒必爭，且堅忍不拔的遊戲精神！

ROG Cetra True Wireless SpeedNova的設計不僅僅是美學上的提升，更體現了實用性與創新性的結合。耳機本體的人體工學設計，佩戴舒適且穩固，即使在長時間使用中也能保持舒適感。耳機的觸控操作區域經過優化，使用更加靈敏且直觀，讓玩家無論在遊戲中或是聆聽音樂時，能夠快速反應，輕鬆操控。同時，耳機的防水性依然維持著實用的IPX4級防水標準，無論是運動還是日常使用，都能應對各種環境挑戰！

/ultra-immersed/

除了外觀設計上的大量改進，ROG Cetra True Wireless SpeedNova在內在的技術規格上也有非常顯著的提升。不但大幅增加電池續航力，讓盒內快速充電僅需5分鐘即可使用1小時，完全充飽電力則可提供長達46小時（藍牙模式）的聆聽時間，還引入了利用LC3+編解碼器提供高解析度24位元96 kHz音訊的ROG SpeedNova無線技術的2.4 GHz無線連接模式，進一步降低遊戲時的音訊延遲，顯著提高了連接時的穩定性；此外，支援同時配對兩部裝置的多點連線模式，確保遊戲連線不中斷，這一創新功能讓玩家在遊戲與其他設備之間無縫切換，提升了使用的便利性和靈活性。

更為突出的是，新加入了骨傳導AI麥克風，讓ROG Cetra True Wireless SpeedNova即使沒有收音桿，也能有效地減少背景噪音，確保清晰的收音品質，這項技術不僅在遊戲對話中表現出色，在日常電話通話中也同樣出色，讓玩家們無論在何種環境下都能享受清晰無干擾的通話品質；而主動降噪功能亦升級為能夠持續偵測耳機的配戴方式，並根據聲音在耳道內的反射情況即時調整，大幅提升降噪效果的自適應主動降噪功能(ANC)；再來，無論是嘈雜的街道或安靜的咖啡廳，智慧自動模式都能識別環境噪音，並自動適應周圍環境，動態調整降噪等級，以提升在安靜環境中使用的舒適性。而這些技術的革新，使得遊戲時的整體體驗更加細緻、完整且沉浸，讓玩家能夠全心投入於遊戲之中，享受每一個精彩時刻！
